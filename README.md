# Système de Réservation de Voyages

## 🌟 Description du Projet
Ce projet vise à moderniser la gestion des réservations d'une agence de voyage grâce à une solution orientée objet (OOP). Il offre une interface intuitive pour gérer les utilisateurs, les activités (vols, hôtels, circuits touristiques) et les réservations.

---

## 🚀 Fonctionnalités Principales

### 🔐 Authentification et Autorisation
- Système sécurisé d'inscription et de connexion.
- Gestion des rôles : Administrateur, Client authentifié, Visiteur.

### 🛠️ Gestion des Utilisateurs et Activités
- **Administrateur :**
  - Créer, modifier ou supprimer des activités.
  - Gérer les utilisateurs et leurs rôles.
  - Visualiser, confirmer ou annuler les réservations.
  - Archiver ou bannir des utilisateurs.
- **Client Authentifié :**
  - Rechercher des activités (vols, hôtels, circuits touristiques, etc.).
  - Réserver, modifier ou annuler des réservations en ligne.
  - Personnaliser des options pour les activités.
- **Visiteur :**
  - Parcourir les activités disponibles sans inscription.
  - S'inscrire pour accéder aux fonctionnalités avancées.
